Cài đặt

Các thư viện quan trọng: Trong quá trình khởi chạy phần mềm, tùy theo hệ điều hành và cấu hình máy, phần mềm có thể yêu cầu các thư viện khác nhau. Chỉ cần bấm Install để tải về thư viện đó.

Các bước khởi chạy:

Bước 1: Tải file nén phần mềm về máy. Khi tải xong thì giải nén file.
Bước 2: Mở thư mục vừa giải nén trên VSCode. File > Open Folder > File đã giải nén.
Bước 3: Run file LoginUI trong mục GUI.
Bước 4: Chọn 1 hình ảnh rõ mặt người để thực hiện quá trình phân tích.
